Оператори spread і rest однакові — це три крапки (...)

Оператор spread використовується для "розпакування" елементів масиву або властивостей об'єкта. Наприклад ситуація: є масив чисел [1, 2, 3], і хочемо додати до нього інші числа. Замість того, щоб додавати кожен елемент окремо, можемо використати spread оператор

const array1 = [1, 2, 3];
const array2 = [...array1, 4, 5, 6];
console.log(array2); 

Виведе [1, 2, 3, 4, 5, 6]

тобто spread оператор "розпаковує" елементи першого масиву і додає їх до нового масиву разом з іншими елементами. 
таким ж чином можна об'єднувати декілька масивів в один.

Оператор rest використовується для збирання залишкових елементів у масив. Наприклад: є функція, яка приймає багато аргументів і просто виводить їх усі разом

function showArguments(...args) {
  console.log(args);
}

showArguments(1, 2, 3); // [1, 2, 3]
showArguments('a', 'b', 'c', 'd'); // ['a', 'b', 'c', 'd']

тобто ...args збирає всі передані аргументи в масив args. Потім ми просто виводимо цей масив у консоль.
